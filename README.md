# SocketPy

A beginner-friendly Python project demonstrating Network Programming using TCP and UDP sockets. This repository provides simple client-server implementations to help understand how computers communicate over a network using Python's built-in `socket` module.

## Overview

SocketPy is designed as an educational project for students and developers who want to learn the fundamentals of network communication. It covers the core concepts of socket programming, including establishing connections, sending and receiving data, and implementing both TCP and UDP communication protocols.

## Features

* TCP Client-Server Communication
* UDP Client-Server Communication
* Simple and Beginner-Friendly Code
* Demonstrates Real-Time Data Exchange
* Easy to Extend into Chat Applications or File Transfer Systems

## Technologies Used

* Python 3
* Socket Programming
* TCP/IP Networking
* UDP Networking



## Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/SocketPy.git
cd SocketPy
```

### Run TCP Example

Start the server:

```bash
python tcp_server.py
```

Open another terminal and run:

```bash
python tcp_client.py
```

### Run UDP Example

Start the UDP server:

```bash
python udp_server.py
```

Open another terminal and run:

```bash
python udp_client.py
```
